## Tools Used

* Mongoose (ODMs are Life)
* Nomics API

## How to use
* set up MongoDB Atlas
* set environment variables, (API_KEY, API_URL, CONNECTION_STRING), create .env file in root folder
* yarn dev in terminal to run locally
